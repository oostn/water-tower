# Water Tower Design Project

A complete 3D water tower design project created using a combination of Autodesk Advanced Steel, SolidWorks, and Autodesk Inventor.  
This project demonstrates structural modelling, mechanical tank design, and industrial pipe routing workflows across multiple CAD platforms.

---

## Project Overview

This project focuses on the design and modelling of a steel water tower structure, including:

- Structural support framework (in progress)
- Water storage tank (not yet modelled)
- Pipework and connections (not yet modelled)
- Industrial assembly workflow

Each software is used for a specific purpose to simulate a real-world multidisciplinary engineering environment.
